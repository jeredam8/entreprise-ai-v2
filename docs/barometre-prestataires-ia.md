# Baromètre des prestataires IA en France

## Objectif

Créer un futur rapport annuel capable de devenir une référence marché sur les prestataires IA en France.

Le baromètre doit analyser :

- les tarifs ;
- les spécialités ;
- les secteurs servis ;
- les budgets moyens ;
- les délais moyens ;
- les outils utilisés ;
- les types de projets les plus demandés.

## Données à collecter

- Type de prestataire.
- Taille de structure.
- Localisation.
- Spécialités.
- Secteurs.
- Budget minimum.
- Tarif jour moyen.
- Types de projets.
- Stack technique.
- Références.
- Volume de missions.
- Délais.
- Satisfaction client.

## Livrables possibles

- Rapport annuel PDF.
- Pages SEO.
- Communiqué de presse.
- Infographies.
- Newsletter.
- Classements.
- Contenus LinkedIn.

## Garde-fous

Les classements doivent afficher leurs critères. Les contenus partenaires doivent être identifiés comme tels. Les données déclaratives doivent être distinguées des données vérifiées.

## Opportunité stratégique

Le baromètre peut créer une boucle d'autorité : les prestataires veulent être recensés, les entreprises cherchent un point de repère, les médias B2B cherchent des données marché.
